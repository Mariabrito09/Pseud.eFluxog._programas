# Lógica de Programação: Pseudocódigos e Fluxogramas 💻

Este repositório contém uma série de exercícios práticos de Lógica de Programação desenvolvidos como parte das atividades do curso técnico em **Desenvolvimento de Sistemas** oferecido pelo **SENAI**. 

O objetivo principal deste projeto é exercitar e demonstrar a estruturação do pensamento lógico e a resolução de problemas através da criação de algoritmos (pseudocódigos) e suas respectivas representações visuais (fluxogramas).

---

## 🛠️ Ferramentas Utilizadas

Para o desenvolvimento das lógicas e diagramas, foram utilizadas as seguintes ferramentas:

* **VisuAlg:** Utilizado para a escrita, estruturação e teste dos algoritmos em pseudocódigo (arquivos com a extensão `.alg`).
* **Draw.io (diagrams.net):** Utilizado para a criação, desenho e exportação dos fluxogramas para facilitar a visualização da lógica (arquivos exportados em `.png`).

---

## 📂 Atividades Desenvolvidas

Abaixo estão listados os exercícios desenvolvidos neste repositório. Cada problema resolvido é composto pelo seu arquivo de pseudocódigo e a imagem do seu fluxograma correspondente:

* **Soma de dois números:** Programa simples para receber dois valores e exibir o total da soma.
    * Arquivos: `Soma dos números.alg` e `Soma de dois números.png`
* **Cálculo de Média:** Algoritmo que calcula a média aritmética de notas.
    * Arquivos: `Média.alg` e `Média.drawio.png`
* **Cálculo de Índice de Massa Corporal (IMC):** Programa que recebe peso e altura para determinar o IMC do usuário.
    * Arquivos: `IMC.alg` e `IMC.png`
* **Conversão de Temperatura:** Algoritmo para converter graus Celsius (ºC) para graus Fahrenheit (ºF).
    * Arquivos: `Converter ºC em ºF.alg` e `Converter ºC em ºF.drawio.png`
* **Cálculo da Área de um Terreno:** Programa focado em geometria básica para calcular a área total de um terreno.
    * Arquivos: `Área do terreno.alg` e `Área terreno .drawio.png`
* **Cálculo da Área de um Trapézio:** Algoritmo para aplicar a fórmula matemática de área de um trapézio.
    * Arquivos: `Área trapézio.alg` e `Area Trapézio.drawio.png`

---

## 🚀 Como visualizar e executar

1.  **Visualizar os fluxogramas:** Basta navegar até os arquivos com a extensão `.png` e abri-los diretamente aqui no GitHub para ver os diagramas.
2.  **Executar os pseudocódigos:** * Faça o download do software gratuito **VisuAlg**.
    * Baixe os arquivos com a extensão `.alg` deste repositório.
    * Abra-os no VisuAlg e pressione `F9` para rodar e testar os algoritmos.
